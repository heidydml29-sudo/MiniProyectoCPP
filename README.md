# Sistema de Registro de Estudiantes

## Descripción
Este programa en C++ permite registrar estudiantes junto con sus notas.
El usuario puede ingresar datos, visualizarlos y realizar cálculos básicos.

## Funcionalidades
- Registrar estudiantes (nombre y nota)
- Mostrar la lista de estudiantes
- Calcular el promedio de notas
- Mostrar la mayor nota (mejora creativa)

## Estructura del programa
El programa está organizado en funciones:
- registrar(): para ingresar datos
- mostrar(): para ver los datos
- promedio(): para calcular el promedio
- mayorNota(): para encontrar la nota más alta

## Tecnologías usadas
- Lenguaje: C++
- Consola
